## Framework version

This working group is pinned to **[`trustx-framework@v1.0.0`](https://github.com/RAI-Institute/trustx-framework/releases/tag/v1.0.0)**. We upgrade deliberately when a new release lands; the decision is recorded in [`meeting-notes/`](./meeting-notes).

Any sector-specific dimensions we add (e.g. settlement reversibility, transaction limits, KYC/AML controls) sit **on top of** the 12 core dimensions and aggregate the same worst-case-wins way — see the framework's [`docs/architecture.md`](https://github.com/RAI-Institute/trustx-framework/blob/main/docs/architecture.md).
